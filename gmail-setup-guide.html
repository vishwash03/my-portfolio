Step 1: Enable 2-Step Verification

1
Go to Google Account Settings
  1. Visit myaccount.google.com
  2. Click Security in the left sidebar
  3. Look for How you sign in to Google
  4. Click 2-Step Verification
2
Follow the Verification Steps
  1. Google will ask for your password (enter it)
  2. Verify using your phone number or backup code
  3. Complete all prompts
  4. You'll see a confirmation message

Step 2: Generate App Password

3
Access App Passwords Settings
  1. Go to myaccount.google.com/apppasswords
  2. You may need to sign in again
  3. Select these options:
    • App: Mail
    • Device: Windows Computer
  4. Click Generate
4
Copy Your App Password

Google will show a 16-character password like this:

ywvi fygj aoyf swtu

‚ö†Ô∏è Copy this password now! It will only show once.

Step 3: Update Server Configuration

5
Edit server.js File
  1. Open the file server.js in your code editor
  2. Find this line (around line 18):
    const EMAIL_PASSWORD = process.env.GMAIL_APP_PASSWORD || 'YOUR_16_CHAR_APP_PASSWORD_HERE';
  3. Replace YOUR_16_CHAR_APP_PASSWORD_HERE with the 16-character password from Step 4
  4. Example:
    const EMAIL_PASSWORD = 'ywvi fygj aoyf swtu';
  5. Save the file

Step 4: Restart Server

6
Restart Node.js Server
  1. Open PowerShell or Terminal
  2. Run this command:
    taskkill /IM node.exe /F && node server.js
  3. You should see: ‚úÖ Email service ready

Security & Privacy

üîí Keep Your App Password Safe
  • Never share your App Password publicly
  • It's different from your main Gmail password
  • You can revoke it anytime in Google Account settings
  • Never commit server.js with real credentials to GitHub (use environment variables instead)

Using Environment Variables (Advanced)

For production, use environment variables instead of hardcoding:
