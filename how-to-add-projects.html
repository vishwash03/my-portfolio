‚ùì FAQ

Q: Why don't I see my new projects immediately?
A: Projects sync automatically every 2 seconds. If you don't see them after 3 seconds, refresh the page (F5) or check that the backend server is running.
Q: Can I edit or delete projects?
A: Yes! In the Project Manager, each project has Edit and Delete buttons in the table. Click to modify anytime.
Q: Do I need to restart the server?
A: No! Changes are saved instantly to the database. The server keeps running and the projects page auto-syncs.
Q: Can I add projects from multiple computers?
A: Yes! As long as they can access your backend server at http://localhost:3000, they can add projects. (For production, you'd deploy the server online.)
Q: What if the backend server crashes?
A: The system has a fallback! It will use localStorage (browser storage) to show the last known projects. Restart the server to sync again.
Q: Can I add videos or other media?
A: Currently, the system supports images only. You can add a link to your project which can contain videos or other media.
